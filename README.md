## 📘 What this is

supply-chain-ai-for-beginners is a guided course for people who want to learn how AI supports supply chain work. It covers the basics first, then moves into real use cases and production steps.

The lessons use clear language and simple examples. You can follow along on a Windows PC with no prior programming experience.

## 🎯 What you will learn

- What AI means in supply chain work
- How machine learning helps with planning and forecasting
- How deep learning fits into supply chain tasks
- How to use Python for simple data work
- How to think about AI projects from start to finish
- How teams move from learning to production
- How to use practical tools for real business cases

## 🖥️ What you need on Windows

Use a Windows 10 or Windows 11 PC with:

- At least 8 GB of RAM
- 5 GB of free disk space
- A stable internet connection
- Microsoft Edge, Chrome, or another modern browser
- A ZIP tool if the files come in a compressed folder
- Python 3.10 or later if you plan to run the lessons locally

If you only want to read the course material, a browser is enough.

## 🧭 How to use it on Windows

### 1. Open the download
Find the file or folder you downloaded from GitHub.

### 2. Extract the files
If the file ends in `.zip`, right-click it and choose Extract All.

### 3. Open the course folder
Open the extracted folder and look for the main lesson files. These may be Markdown files, notebooks, or folders named by lesson number.

### 4. Start with Lesson 1
Begin with the first lesson and move in order. The course builds from basic ideas to full project work.

### 5. Follow each lesson
Read the lesson text, look at the examples, and complete any exercises in the same order they appear.
